    一、创建 Plugin Module

    在 Android Studio 中通过如下步骤新建一个 Gradle Plugin Module：

    1. 在 Android Project (已有或者新建) 中新建一个用于开发 Gradle 插件的 Module : plugin1，可以随意选择 Module 类型，注意：此 Module 只是我们开发插件的壳，后续会删除大部分内容，可以理解为与 Module 类型相关的配置都会被删除。
    2. 将新建的 Module 里面的内容删除，只保留如下文件：
      • build.gradle 文件
      • src/main 目录

    3. 修改 build.gradle 文件。将 build.gradle 文件修改成插件配置样式，修改后如下：

    4. 在 src/main 目录下新建 groovy 目录即包名用于插件开发。Gradle 是基于 groovy 开发的，因此 Gradle 插件相当于一个 groovy 项目。

    5. 新建插件类。在 groovy 目录下新建 groovy 文件， New -> File -> MyPlugin.groovy ；插件同时还需要实现 org.gradle.api.Plugin 接口。最终文件如下所示：

    6. 声明 Gradle 插件类。在 src/main 目录下新建 resources 目录，然后在 resources 目录里新建 META-INF 目录，接着在 META-INF 目录里新建 gradle-plugins 目录，最后在 gradle-plugins 目录里新建 MyPlugin1.properties 文件，并在 MyPlugin1.properties 文件中指明插件的类。其中，文件名前缀 MyPlugin1 即为插件引用名，比如，插件命名 MyPlugin1.properties 时使用插件写法为 apply plugin:MyPlugin1。最终配置文件如下：

    三、使用自定义插件

    当需要使用其他插件的时候主要有两个步骤：

    1. 添加插件引用来源的仓库地址(本地、Maven仓库地址)及项目依赖
    2. 声明引用插件，语法为 apply plugin: 'plugin name'

    通常，我们在 根目录 build.gradle 文件下配置 步骤1：仓库及依赖配置，在需要使用插件的 Module 中 build.gradle 中配置 步骤2：声明插件。
